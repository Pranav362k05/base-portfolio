# Portfolio Website

This is a personal portfolio website built using HTML, CSS, and JavaScript.  
It follows a clean, structured layout inspired by encyclopedia-style design, focusing on readability, accessibility, and organization.

## Features

- Structured multi-section layout (Introduction, Skills, Projects, etc.)
- Sticky Table of Contents with active section highlighting (Scroll Spy)
- Responsive 3-column design (TOC + Content + Infobox)
- Profile infobox with images and key details
- Smooth scrolling navigation
- Accessibility features (skip link, semantic HTML)

## Tech Stack

- HTML5
- CSS3 (Grid, responsive design, custom properties)
- JavaScript (Intersection Observer API)
